# homebrew-jmax

Homebrew tap for [JMax](https://charlot-lang.dev), a math-native language and CLI for science, data, and simulation.

## Install

```sh
brew install openie-dev/jmax/jmax
```

Or tap first, then install:

```sh
brew tap openie-dev/jmax
brew install jmax
```

## Verify

```sh
jmax --version
jmax eval "[[1,2],[3,4]] * [5,6]"
```

## Platforms

Prebuilt binaries for macOS (Apple Silicon and Intel) and Linux x86-64. Windows binaries and the browser playground are on the [releases page](https://github.com/openIE-dev/jmax/releases) and [play.charlot-lang.dev](https://play.charlot-lang.dev).
